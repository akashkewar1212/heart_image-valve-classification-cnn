# Heart Image Classification (HOG+SVM, Transfer Learning)

## Overview
This project builds image classification models to:
1. Classify **Heart vs Non-Heart** images using **HOG feature extraction + SVM**
2. Classify **Open vs Closed** heart valve images using **HOG + SVM**
3. Improve performance using **data augmentation + transfer learning** (TensorFlow)

## Tech Stack
- Python
- OpenCV
- scikit-image (HOG)
- scikit-learn (SVM)
- Matplotlib
- TensorFlow + TensorFlow Hub


## Results
- Task 1 (Heart vs Non-Heart): Accuracy = **100%**
- Task 2 (Open vs Closed): Accuracy = **100%**
- Task 3 (Transfer Learning): Accuracy = **96%**
